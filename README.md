# Refactoring UI Example

一个用 React、Vite 和 Tailwind CSS 构建的交互式示例站，用来展示《Refactoring UI》里的常见界面设计原则。

页面围绕视觉层级、留白间距、色彩系统、字体排印、光影层次和细节打磨展开，每个部分都提供「优化前 / 优化后」对比，方便直观看到设计调整带来的变化。

## 技术栈

- React 19
- TypeScript
- Vite
- Tailwind CSS
- vite-plugin-singlefile

## 本地开发

安装依赖：

```bash
npm install
```

启动开发服务器：

```bash
npm run dev
```

构建生产版本：

```bash
npm run build
```

预览构建结果：

```bash
npm run preview
```

## 说明

这是一个学习用途的非官方示例项目，核心设计理念来自 Adam Wathan 和 Steve Schoger 的《Refactoring UI》。
